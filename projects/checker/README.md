# Шашки

Классическая настольная игра в шашки с современным деловым дизайном.

## Особенности

- **Три типа правил**: Русские, международные и английские шашки
- **Два режима игры**: Игрок против игрока или игра против компьютера
- **Интеллектуальный AI**: Три уровня сложности (легкий, средний, сложный)
- **Анимации**: Плавные анимации движения шашек и взятия фигур
- **Звуковые эффекты**: Звуки для ходов, взятия и победы
- **Таймер**: Настраиваемый таймер для каждого игрока
- **Сохранение прогресса**: Автоматическое сохранение игры в localStorage
- **Статистика**: Учет побед, поражений и ничьих
- **История ходов**: Полная история с нотацией
- **Адаптивный дизайн**: Работает на всех устройствах

## Правила игры

### Русские шашки
- Доска 8x8
- Дамки могут ходить на любое расстояние по диагонали
- Обязательное взятие

### Международные шашки
- Доска 10x10
- Дамки могут ходить на любое расстояние
- Обязательное взятие с выбором максимального количества

### Английские шашки (Чекерс)
- Доска 8x8
- Дамки ходят только на одну клетку
- Простые правила для начинающих

## Технологии

- React + TypeScript
- Tailwind CSS
- Framer Motion (анимации)
- Web Audio API (звуки)
- LocalStorage (сохранение)

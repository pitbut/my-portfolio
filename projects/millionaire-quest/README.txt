  💰 КАК СТАТЬ МИЛЛИАРДЕРОМ - Квест с пазлами

📋 ОПИСАНИЕ
-----------
Интерактивный квест с механикой сборки пазлов, поиска меток на изображениях
и составления финального пароля из найденных букв.

🎯 КАК ИГРАТЬ
-------------
1. Соберите пазл из частей (перетаскивайте кусочки)
2. Нажмите на красную точку (метка с буквой)
3. Введите найденную букву/цифру/слово
4. Повторите для всех уровней
5. Составьте финальный пароль из букв
6. Найдите сокровище!

✨ ВОЗМОЖНОСТИ
--------------
✓ Создание квестов с пазлами
✓ Вопросы и задания для каждого уровня
✓ Отметки мест на изображениях
✓ Ввод букв/цифр/слов
✓ Защита паролем + секретный вопрос
✓ Импорт/экспорт JSON
✓ LocalStorage для хранения
✓ Мобильная версия (drag & touch)

📝 СОЗДАНИЕ КВЕСТА
------------------
1. Откройте "Создать новый квест"
2. Заполните название и финальный пароль
3. Добавьте уровни:
   - Загрузите изображение
   - Кликните на месте с буквой (красная метка)
   - Введите букву/цифру/слово
   - Добавьте вопрос или задание (опционально)
4. Загрузите фото сокровища
5. Установите пароль для защиты (опционально)
6. Сохраните!

🔒 ЗАЩИТА
---------
- Пароль для редактирования (SHA-256 хеш)
- Секретный вопрос для восстановления
- Импорт/экспорт с сохранением защиты

📤 ИМПОРТ/ЭКСПОРТ
-----------------
ЭКСПОРТ: Кнопка "Экспортировать JSON" → сохранить .json файл
ИМПОРТ: Кнопка "Загрузить квест" → выбрать .json файл

ФОРМАТ JSON:
{
  "title": "Название",
  "author": "Автор",
  "correctPassword": "ФИНАЛЬНЫЙ_ПАРОЛЬ",
  "passwordHash": "хеш_для_редактирования",
  "secretQuestion": "Секретный вопрос?",
  "secretAnswerHash": "хеш_ответа",
  "levels": [
    {
      "image": "base64_строка",
      "letter": "Б",
      "markerX": 50,
      "markerY": 50,
      "question": "Найдите на фото... (опционально)"
    }
  ],
  "treasureLocation": {
    "image": "base64_строка",
    "markerX": 50,
    "markerY": 50
  }
}

📱 МОБИЛЬНАЯ ВЕРСИЯ
-------------------
- Touch drag & drop для пазлов
- Оптимизированные размеры
- Адаптивный дизайн
- Работает на iOS и Android

⚙️ ТЕХНИЧЕСКИЕ ДЕТАЛИ
----------------------
- HTML5 + CSS3 + JavaScript ES6+
- Web Crypto API (хеширование)
- LocalStorage (хранение)
- Canvas API (сжатие изображений)
- Touch Events (мобильная версия)

💡 СОВЕТЫ
---------
- Используйте четкие изображения
- Метки ставьте в заметных местах
- Финальный пароль = все буквы подряд
- Экспортируйте квесты для резервной копии
- Запомните пароль редактирования!

🐛 РЕШЕНИЕ ПРОБЛЕМ
------------------
Пазл не собирается: перетаскивайте кусочки друг на друга
Буква не принимается: проверьте регистр (все ЗАГЛАВНЫЕ)
Забыли пароль: используйте секретный вопрос
Квест не загружается: проверьте формат JSON файла

🚀 БЫСТРЫЙ СТАРТ
════════════════════════════════════════════════════════════════════

1️⃣ ЗАПУСК:
   Откройте: index.html в браузере
   
2️⃣ ПОПРОБУЙТЕ ДЕМО:
   • Нажмите "🎮 Пройти квест"
   • Выберите "💰 Как стать миллиардером"
   • Пройдите 6 уровней бизнес-викторины

3️⃣ СОЗДАЙТЕ СВОЙ КВЕСТ:
   • Нажмите "✏️ Создать новый квест"
   • Заполните информацию
   • Установите пароль (необязательно)
   • Добавьте уровни с вопросами
   • Сохраните!

════════════════════════════════════════════════════════════════════

🔒 БЕЗОПАСНОСТЬ
════════════════════════════════════════════════════════════════════

ПАРОЛЬ:
• Хешируется через SHA-256
• Защищает квест от редактирования
• Нельзя восстановить (только через секретный вопрос)

СЕКРЕТНЫЙ ВОПРОС:
• Резервный способ восстановления доступа
• Ответ также хешируется
• Задаётся при создании квеста

💡 ВАЖНО: Запомните пароль и ответ на секретный вопрос!

════════════════════════════════════════════════════════════════════

💾 ЭКСПОРТ И ОБМЕН
════════════════════════════════════════════════════════════════════

ЭКСПОРТ КВЕСТА:
1. Откройте редактор с вашим квестом
2. Нажмите "📤 Экспортировать JSON"
3. Файл сохранится на компьютер

ИМПОРТ КВЕСТА:
1. Главное меню → "📥 Загрузить квест (JSON)"
2. Выберите .json файл
3. Квест добавится в коллекцию

ОБМЕН С ДРУЗЬЯМИ:
• Отправьте JSON файл
• Получатель импортирует его
• Пароль защитит от редактирования

🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ
════════════════════════════════════════════════════════════════════

ТЕХНОЛОГИИ:
• HTML5 (семантическая разметка)
• CSS3 (Flexbox, Grid, Animations)
• JavaScript ES6+ (модули, async/await)
• Web Crypto API (хеширование)
• LocalStorage API (хранение)
• Canvas API (сжатие изображений)

ТРЕБОВАНИЯ:
• Современный браузер (Chrome 90+, Firefox 88+, Safari 14+)
• JavaScript включён
• LocalStorage доступен

ОГРАНИЧЕНИЯ:
• LocalStorage: ~5-10 МБ
• Изображения: автосжатие до 1200px
• Варианты ответов: максимум 6
